## CSS Flexbox Properties Explained

### Body (`d-flex flex-column min-vh-100`)
```css
display: flex;           /* Make it a flex container */
flex-direction: column;  /* Stack children vertically */
min-height: 100vh;       /* At least full viewport height */
```

### Main Content (`flex-fill`)
```css
flex: 1 0 auto;  /* Grow to fill space, don't shrink */
```
- **1** = flex-grow (expands to take available space)
- **0** = flex-shrink (doesn't shrink)
- **auto** = flex-basis (natural size)

### Footer (`mt-auto`)
```css
margin-top: auto;    /* Pushes footer to bottom */
flex-shrink: 0;      /* Prevents footer from shrinking */
```

## Advantages of This Approach

? **No JavaScript** - Pure CSS solution  
? **Responsive** - Works on all screen sizes  
? **Bootstrap native** - Uses built-in utilities  
? **Simple** - Easy to understand and maintain  
? **Flexible** - Content can grow or shrink  
? **No fixed heights** - Footer height can vary  

## Alternative Approaches (Not Used)

### 1. Position: fixed (? Not ideal)
```css
footer {
  position: fixed;
  bottom: 0;
}
```
**Problem:** Footer overlaps content, requires extra padding

### 2. Negative margin (? Old technique)
```css
body { margin-bottom: -60px; }
footer { height: 60px; }
```
**Problem:** Requires fixed footer height, not flexible

### 3. Calc() height (? Complex)
```css
main { min-height: calc(100vh - header - footer); }
```
**Problem:** Requires knowing exact heights, not responsive
